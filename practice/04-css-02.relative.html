球场自然轨迹

这是一个发球，球按照正常轨迹在场地线上滚动。

它只是随着重力，沿着预设的路径向前。

这个球是场地的一部分，随着场地一起移动。

你无法从正常轨迹中将其分离，它就是球场的一部分。

我们关注的是球落点的位置。

球员的微调

比赛精彩！球员正在准备接发球。

他根据球的落点，进行微小的步法调整。

这个球稍微向后移动了80像素，就像接球前球员的小碎步侧移。

你看，它并没有脱离整体的流畅运动，只是在自己的“正常位置”上做了些微调。

裁判只是微微注意到这个调整。

.relative { position: relative; top:80px;}

特殊控制的球

突然！一个未经允许的球被扔进了场地！

它被精确地定在了场地中央，脱离了正常滚动。

这个球被直接放置在距离场地顶部120像素，左侧350像素的位置。

它不再受其他球的影响，也不会影响场地的自然流程。它完全是“自己”的。

它被特殊管理，不属于球场上的任何一个正常轨迹。

.absolute { position: absolute; top:120px; left:350px;}
